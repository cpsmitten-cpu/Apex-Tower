Visitor guide

Simple details for a smooth day—parking, Wi-Fi, printing, and after-hours access.

Arrival & access

  • Guest Wi-Fi: Apex_Guest · password SummitView2026!
  • Parking validation: Main St. Garage—validate at the front desk before leaving
  • Overflow parking: 300 W 4th Street (~5 min walk)
  • After-hours: main doors lock at 6:00 PM; escort needed to exit via night door

Inside the tower

  • Restrooms: every floor (central core); plus single-stall options (East Wing)
  • Printing: email docs to print@apextower.com for a 6-digit release code
  • Room availability: check Room Wizard tablets outside each door
  • Emergency exits: Stairwells A or B—do not use elevators

Note: All details are fictional for this demo landing page.

FAQ

Quick answers that keep the day moving.

What’s the Wi-Fi password? ⌄
Connect to Apex_Guest with password SummitView2026!, then accept the terms on the splash page.
Do you validate parking? ⌄
Yes—parking is validated for the Main St. Garage. Present your ticket at the front desk before you leave.
Is there EV charging? ⌄
Yes—ChargePoint stations on Level P2, Row C. First-come, first-served.
Where can I take a quiet call? ⌄
Soundproof phone booths are located along the windows in the North Corridor. First-come, first-served.
How do I print something? ⌄
Email your document to print@apextower.com. You’ll receive a 6-digit release code to use at any printer.
What time does the building close? ⌄
Main doors lock at 6:00 PM. If you’re inside after hours, you’ll need an escort to exit via the night door.
